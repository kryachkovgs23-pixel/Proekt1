Понятие информации и информационные процессы

Информация - это сигналы, получаемые человеком с помощью органов чувств и различных приборов.

Информацию можно классифицировать:
- по способу восприятия: на визуальную, аудиальную, тактильную, обонятельную и вкусовую.


Вид информации Органы чувств Пример
Визуальная Зрение С помощью зрения мы рассматриваем картины, читаем книги
Аудиальная Слух С помощью слуха воспринимаем музыку, звуки машин
Тактильная Осязание С помощью кожи чувствуем холодную стену, гладкость поверхности стола
Вкусовая Вкус С помощью рецепторов на языке мы узнаём, какое мороженое вкусное
Обонятельная Обоняние С помощью обоняния мы ощущаем запахи

Объективность - информация не зависит от мнения её получателя.
Достоверность - отражает истинное положение дел.
Полнота - объём информации достаточен для решения задач.
Актуальность - информация доставлена вовремя.
Полезность - нужна для принятия решения (решения задач).
Понятность - выражена на языке, понятном тому, для кого она предназначена.
